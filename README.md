# ✨ Áurea Joias | Elegância Atemporal

Este projeto é uma **landing page de alta costura e joalheria**, desenvolvida como um demonstrativo técnico de habilidades em **Front-end**.  

O foco principal foi a criação de uma interface sofisticada, com tipografia refinada e uma experiência de usuário que remete ao mercado de luxo.

🔗 **Site online:**  
👉 https://aureajoias.netlify.app/

---

## 🚀 Tecnologias e Implementação

O projeto foi construído do zero, priorizando **código limpo** e uma **estética visual impecável**.

### 🏗️ Arquitetura

**HTML5 Semântico**  
Estrutura organizada garantindo acessibilidade e base sólida para SEO em e-commerce.

### 🎨 Estilização

**CSS3 puro**, utilizando o conceito de **Design System** com variáveis globais para:

- Paleta de cores (tons dourados e nudes)
- Tipografia refinada
- Espaçamentos consistentes

### ⚡ Interatividade

**JavaScript Vanilla** para comportamentos dinâmicos como:

- Cabeçalho fixo
- Controle de estados de componentes
- Efeitos de hover refinados
- Transições suaves

### 📱 Design Responsivo

Layout totalmente adaptável, mantendo elegância tanto em **desktops de alta resolução** quanto em **dispositivos móveis**.

---

## 📱 Diferenciais do Projeto

- ✨ **Estética Minimalista**  
  Uso estratégico de espaços em branco e tipografia clássica para destacar os produtos.

- 🧱 **Componentização Manual**  
  Cards de produtos, seções de coleções e áreas de contato construídos sem bibliotecas externas, demonstrando domínio de **CSS Grid** e **Flexbox**.

- 🚀 **Otimização de Assets**  
  Uso estratégico de imagens e ícones para garantir carregamento fluido.

- 👑 **UI/UX Premium**  
  Hierarquia visual clara e navegação guiada pelas coleções:
  - Eternidade
  - Esplendor
  - Glamour
